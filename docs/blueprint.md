# **App Name**: Face Fortune

## Core Features:

- Image Analysis: Analyze the uploaded image to identify key facial features using AI and facial recognition.
- Face Reading Report: Generate a detailed face reading report based on the principles of Chinese face reading, including assessments of five elements, spirit, twelve palaces, and health.
- Markdown Display: Display the face reading report in a clear, well-formatted Markdown layout, emphasizing key findings and recommendations.
- Personalized Advice: Provide personalized advice based on the analysis, covering aspects like career, wealth, relationships, and health improvements.
- Image Upload: Allow users to upload their face images for analysis.
- Lucky Charms Tool: Provide customized lucky charm suggestions generated by an AI tool based on the analysis, each intended to promote positive qualities that can improve work-life.

## Style Guidelines:

- Primary color: Earthy gold (#B8860B) to invoke traditional wisdom.
- Background color: Soft beige (#F5F5DC) to create a neutral, inviting canvas.
- Accent color: Deep maroon (#800000) to add richness and highlight key elements.
- Body font: 'Literata', a serif font that adds a vintage and literary feeling. Note: currently only Google Fonts are supported.
- Headline font: 'Playfair', a serif font with an elegant, fashionable, high-end feel. Note: currently only Google Fonts are supported.
- Use simple, classic icons representing different face reading aspects. Should invoke simplicity and heritage.
- Clean, structured layout with a clear visual hierarchy to present analysis results effectively. Place upload image to the top for an instant preview.
